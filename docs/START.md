# 项目初始想法

本项目用于饥荒联机版维基的维护

## 功能划分

### 游戏代码解析
饥荒联机游戏代码是一个scripts.zip压缩包，通过解析里面文件实现以下功能:
- 解析po文件获取游戏内实体代码-英文名-中文名的映射关系，组织成结构体
- 解析recipes.lua文件获取游戏内所有配方，组织成结构体
- 解析po文件里人物对实体的检查台词，组织成便于检索的结构体
- 解析skilltree相关的文件，将其中的技能树定义组织成结构体
- 解析prefabs/目录下的文件，通过语法树解析，将其中的预制体定义的setPrefabNameOverrides解析出来，统计出预制体的名称和重定向的代码名称

### 维基维护
实现维基相关的以下功能
- 基本的账号登录
- 维基页面的编辑接口
- 维基的wikitext的解析
- 维基wikitext的下载和增量更新

### 维护工具
通过将以上两个功能模块整合起来，实现维基的维护工具
- 自动维护相关的模块数据
- 自动维护相关的数据库并且能实现版本比较
- 经过对wikitext的解析，实现对一些特征搜索
